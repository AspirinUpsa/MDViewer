# 📖 POWer Markdown Viewer

Десктопное приложение для просмотра Markdown и текстовых файлов на базе Qt Framework.

![Qt](https://img.shields.io/badge/Qt-5.14+-blue)
![C++](https://img.shields.io/badge/C++-17-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)

---

## ✨ Возможности

- 📄 **Просмотр Markdown** — рендеринг через `QTextDocument::setMarkdown()` (Qt 5.14+)
- 🎨 **Две темы** — светлая и тёмная
- 📖 **Режим чтения** — приятная цветовая схема с засечками (включён по умолчанию)
- 🔍 **Поиск по тексту** — быстрый поиск с подсветкой всех совпадений
- 🔎 **Масштабирование** — увеличение/уменьшение шрифта
- 🖱️ **Drag-and-Drop** — откройте файл перетаскиванием
- 📋 **Копирование** — правая кнопка мыши копирует выделенный текст
- 📂 **Поддержка форматов** — `.md`, `.markdown`, `.txt`

## 📦 Сборка

```bash
# Генерация Makefile
qmake mdviewer.pro

# Сборка (Windows + MinGW)
mingw32-make

# Сборка (Windows + MSVC)
nmake

# Сборка (Linux/macOS)
make
```

### Развёртывание (Windows)

```bash
# Копирование необходимых DLL Qt
windeployqt MDViewer.exe
```

## 🚀 Использование

### Открытие файла

| Способ | Действие |
|--------|----------|
| **Drag-and-Drop** | Перетащите файл на окно программы |
| **Кнопка +** | Нажмите на главный экран |
| **Меню** | Файл → Открыть MD файл |
| **Горячие клавиши** | `Ctrl+O` |

### В окне просмотра

| Элемент | Действие |
|---------|----------|
| **🌙 / ☀️** | Переключение темы |
| **📖 / ❌** | Вкл/выкл режима чтения |
| **A+ / A- / A** | Масштабирование шрифта |
| **🔍** | Поиск с подсветкой всех совпадений |
| **Правая кнопка мыши** | Копировать выделенный текст |

---

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+O` | Открыть файл |
| `F1` | Открыть справку |
| `Ctrl+C` | Копировать выделенное |
| `Esc` | Закрыть окно просмотра |

## 🎨 Темы оформления

### Светлый режим чтения (по умолчанию)
- Фон: тёплый кремовый (#f5f0e6)
- Текст: тёмно-серый (#2c2c2c)
- Шрифт: Georgia, Times New Roman

### Тёмный режим чтения
- Фон: тёмно-серый (#2d313a)
- Текст: светлый (#c9d1d9)
- Шрифт: Georgia, Times New Roman
