# POWer Markdown Viewer — Справка

## Обзор

**POWer Markdown Viewer** — программа для чтения Markdown-файлов с поддержкой тем, поиска и масштабирования.

---

## Возможности

### 📄 Просмотр Markdown
- Рендеринг через `QTextDocument::setMarkdown()`
- Поддержка всех элементов Markdown
- Переход по внешним ссылкам

### 🎨 Темы
- **Светлая** — классический белый фон
- **Тёмная** — тёмный фон для работы вечером
- **📖 Режим чтения** — приятная цветовая схема с засечками (включена по умолчанию)

### 🔍 Поиск
Поиск по тексту в окне просмотра (регистрозависимый).

### 🔎 Масштабирование
- **A+** — увеличить шрифт
- **A-** — уменьшить шрифт
- **A** — сбросить масштаб

### 🖱️ Drag-and-Drop
Перетащите `.md` файл на окно программы для открытия.

### 📋 Копирование
- **Правая кнопка мыши** — копировать выделенный текст
- **Ctrl+C** — стандартное копирование

---

## Как использовать

### Открытие файла

| Способ | Действие |
|--------|----------|
| **Кнопка +** | Нажмите на главный экран |
| **Меню** | Файл → Открыть MD файл |
| **Горячие клавиши** | `Ctrl+O` |
| **Drag-and-Drop** | Перетащите файл на окно |

### В окне просмотра

| Кнопка | Действие |
|--------|----------|
| **🌙 / ☀️** | Переключение темы |
| **❌** | Выход из режима чтения (включен по умолчанию) |
| **📖** | Включить режим чтения |
| **A+ / A- / A** | Масштабирование |
| **Поиск** | Поиск по тексту |
| **Правая кнопка** | Копировать выделенное |

---

## Режим чтения

**Включен по умолчанию** — приятная цветовая схема с засечками:

### Светлый режим чтения
- Фон: тёплый кремовый (#f5f0e6)
- Текст: тёмно-серый (#2c2c2c)
- Шрифт: с засечками (Georgia)
- Увеличенные отступы

### Тёмный режим чтения
- Фон: тёмно-серый (#2d313a)
- Текст: светлый (#c9d1d9)
- Шрифт: с засечками
- Сниженная нагрузка на глаза

---

## Форматы файлов

Поддерживаются файлы:
- `*.md`
- `*.markdown`

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+O` | Открыть файл |
| `F1` | Открыть справку |
| `Ctrl+C` | Копировать выделенное |
| `Esc` | Закрыть диалог |

## Советы

> **💡 Совет:** Режим чтения включён по умолчанию — используйте его для комфортного длительного чтения.

> **💡 Совет:** Нажмите правую кнопку мыши на выделенном тексте для быстрого копирования.

> **💡 Совет:** Перетащите файл прямо из проводника — это быстрее, чем использовать меню.
