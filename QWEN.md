# MDViewer — Контекст проекта

## Обзор проекта

**POWer Markdown Viewer** — десктопное приложение для просмотра Markdown и текстовых файлов, разработанное на **Qt Framework** (C++). Приложение предоставляет удобный интерфейс для чтения MD-файлов с поддержкой тем, поиска, масштабирования и drag-and-drop.

### Основные возможности

- 📄 Рендеринг Markdown через `QTextDocument::setMarkdown()` (Qt 5.14+)
- 🎨 Две темы оформления (светлая/тёмная) + режим чтения с приятной цветовой схемой
- 🔍 Поиск по тексту документа
- 🔎 Масштабирование шрифта (A+, A-, сброс)
- 🖱️ Drag-and-drop для открытия файлов
- 📋 Копирование выделенного текста правой кнопкой мыши
- 📂 Поддержка файлов `.md`, `.markdown`, `.txt`

## Сборка и запуск

### Требования

- **Qt Framework** 5.14+ (для `QTextDocument::setMarkdown()`)
- **C++17**-совместимый компилятор
- **qmake** (входит в состав Qt)

### Сборка

```bash
# Генерация Makefile
qmake mdviewer.pro

# Сборка проекта
make

# Для Windows с MinGW
mingw32-make

# Для Windows с MSVC
nmake
```

### Запуск

```bash
# После сборки
./MDViewer           # Linux/macOS
MDViewer.exe         # Windows
```

### Развёртывание (Windows)

```bash
# Копирование необходимых DLL Qt
windeployqt MDViewer.exe
```

## Технические детали

### Классы

#### `MainWindow`
Главное окно приложения:
- Стартовый экран с кнопкой "+" и drag-and-drop зоной
- Меню "Файл" (Открыть, Выход) и "Справка" (Помощь, Краткая помощь, О программе)
- Управление жизненным циклом `HelpDialog`
- Обработка `dragEnterEvent` и `dropEvent`

#### `HelpDialog`
Диалог просмотра файлов:
- `CopyTextBrowser` — кастомный `QTextBrowser` с копированием по правой кнопке
- `QToolBar` с инструментами:
  - Поиск (QLineEdit)
  - Переключатель темы (🌙/☀️)
  - Режим чтения (📖/❌)
  - Масштабирование (A+, A-, A)
- Методы применения тем: `applyLightTheme()`, `applyDarkTheme()`, `applyLightReadingMode()`, `applyDarkReadingMode()`

#### `CopyTextBrowser`
Наследуется от `QTextBrowser`:
- Переопределяет `mousePressEvent()` для обработки правой кнопки
- Копирует выделенный текст при клике правой кнопкой
- Отключает стандартное контекстное меню (`Qt::NoContextMenu`)

### Цветовые схемы

#### Светлый режим чтения (по умолчанию)
- Фон: `#f5f0e6` (тёплый кремовый)
- Текст: `#2c2c2c` (тёмно-серый)
- Шрифт: Georgia, Times New Roman (с засечками)
- Панель: `#e8e0d5`

#### Тёмный режим чтения
- Фон: `#2d313a` (тёмно-серый)
- Текст: `#c9d1d9` (светлый)
- Шрифт: Georgia, Times New Roman (с засечками)
- Панель: `#21262d`

## Стиль кода и соглашения

- **Язык**: C++17
- **Стиль именования**: CamelCase для классов и методов (Qt-стиль)
- **Префиксы**: `m_` для членов класса (например, `m_textBrowser`)
- **Умные указатели**: Не используются, управление памятью через Qt Parent-Child
- **Сигналы/слоты**: Qt Meta-Object System для связей между компонентами
- **Документация**: Doxygen-стиль комментарии в заголовочных файлах

## Известные особенности

1. **Окно просмотра открывается справа от главного окна** — при загрузке файла главное окно скрывается, показывается диалог
2. **При закрытии диалога главное окно показывается снова** — связь через сигнал `finished`
3. **Режим чтения включён по умолчанию** — приятная цветовая схема с засечками
4. **Правая кнопка мыши копирует выделенное** — вместо контекстного меню
